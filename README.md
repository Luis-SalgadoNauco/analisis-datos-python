\# Análisis de Datos con Python – Día 1



\## Descripción

Proyecto introductorio de análisis de datos usando Python.

Corresponde al Día 1 del módulo de Python dentro del curso de Análisis de Datos.



\## Objetivos

\- Configurar entorno de análisis de datos

\- Usar entornos virtuales en Python

\- Instalar librerías esenciales

\- Ejecutar un primer análisis y visualización



\## Tecnologías utilizadas

\- Python 3.14

\- NumPy

\- Pandas

\- Matplotlib

\- Jupyter Notebook
